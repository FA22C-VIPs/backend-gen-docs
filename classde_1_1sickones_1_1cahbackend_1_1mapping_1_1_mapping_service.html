CardSetEntity de.sickones.cahbackend.mapping.MappingService.mapAddCardSetDtoToCardSetEntity ( AddCardSetDto  dto)

Maps an AddCardSetDto object to a CardSetEntity object.

This method takes an AddCardSetDto object containing data for creating a card set and maps it to a CardSetEntity object. It retrieves the owner entity from the PlayerService based on the provided owner ID, and fetches the corresponding white and black cards using the provided card IDs.
